# AI Security Prompts

A curated collection of **AI prompts designed to help secure Microsoft 365 environments and enterprise systems**.

This repository provides structured prompts that can be used with AI assistants (ChatGPT, Copilot, Claude, etc.) to guide administrators, security engineers, and IT leaders through **real-world security tasks, investigations, and architecture design**.

The goal is to provide **clear, actionable prompts** that generate reliable responses aligned with **security best practices and Microsoft guidance**.

---

# Purpose

Modern IT teams increasingly use AI assistants to:

- troubleshoot systems
- design security architectures
- automate documentation
- investigate security events

However, **poor prompts produce poor answers**.

This repository helps solve that problem by providing **well-structured prompts specifically designed for security operations and Microsoft 365 environments**.

These prompts guide AI tools to provide:

- accurate technical explanations
- step-by-step configuration guidance
- security best practices
- architecture recommendations

---

# Focus Areas

The prompts in this repository primarily focus on:

- Microsoft 365 Security
- Microsoft Entra ID (Azure AD)
- Conditional Access
- Microsoft Defender
- Exchange Online Security
- Identity and Access Management
- Zero Trust Architecture
- Security Operations
